# 🧑‍💻 Aplikasi User Management (Dart)

## 📘 Deskripsi
Ini merupakan aplikasi **User Management** yang dibuat menggunakan bahasa **Dart** dan sudah menerapkan konsep **Object-Oriented Programming (OOP)**.  
Aplikasi ini berjalan di terminal (console) dan memiliki beberapa fitur untuk mengelola data user (Admin dan Customer).

🧩 Fitur Aplikasi

1️⃣ Login User
Ketik angka 1 dan tekan Enter untuk masuk ke fitur login.
Kamu akan diminta untuk memasukkan email dan password.
Jika data benar, sistem akan menampilkan pesan berhasil login.
Jika salah, akan muncul pesan gagal login.

2️⃣ Cari User Berdasarkan Email
Ketik angka 2 dan tekan Enter untuk masuk ke fitur ini.
Masukkan alamat email user yang ingin dicari.
Jika ditemukan, sistem akan menampilkan data lengkap user tersebut.

3️⃣ Tampilkan Semua User Urut Berdasarkan Email
Ketik angka 3 dan tekan Enter untuk menampilkan semua user (Admin dan Customer) yang sudah diurutkan berdasarkan alamat email.

4️⃣ Cari Admin Berdasarkan Permission
Ketik angka 4 dan tekan Enter untuk masuk ke fitur ini.
Masukkan permission yang ingin dicari (misalnya: add, edit, view, atau delete).
Sistem akan menampilkan daftar admin yang memiliki permission tersebut.

5️⃣ Cari Customer Berdasarkan Kota/Daerah
Ketik angka 5 dan tekan Enter untuk masuk ke fitur ini.
Masukkan nama kota atau daerah (contoh: Bandung).
Sistem akan menampilkan semua customer yang berasal dari kota atau daerah tersebut.
